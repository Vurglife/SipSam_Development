General Rules
Players
•	4 players including a Banker/Dealer who the other players bet against.
o	All players are necessary.  When human players are not available to compliment the 4 players, computer simulated players must be available.
•	In this version the objective is to beat the Banker/Dealer. 
Game Play
Board Setup
Each player has his own section with the following features:
1.	3 piles for hands:
a.	Pile for 1st hand (accommodates only 3 cards)
b.	Pile for 2nd hand (accommodates 5 cards)
c.	Pile for 3rd hand (accommodates 5 cards)
These will be placed one above the other moving further away from the player. The 1st hand will be placed furthest, the 2nd hand will be placed in the middle, and the 3rd hand will be placed closest to the player.
2.	1 pile for Special. All thirteen cards will be accommodated in this pile
3.	Place for placing bet. The player could double their bet before the round. If they choose to do this then the banker must decide on accepting the challenge. 
Dealing
•	The Banker is determined by a random draw. 
•	Banker/Dealer cuts the deck, stacks together and flips the top card. 
•	The value of the card determines who is dealt the 1st card.
•	Regardless of the suit, the following applies going around the table from left to right from the Banker’s perspective:
o	1, 5, 9, 13 starts with the Banker
o	2, 6, 10 starts with Player1
o	3, 7, 11 starts with Player2
o	4, 8, 12 starts with Player3.
•	Each player is dealt 13 cards one at a time going around the table from left to right.
•	The cards are dealt face-up but visible only to the player that receives them. No player shall be able to see another player’s card until the turn is ended after 60 seconds. This 60 seconds represent the play time for a player to evaluate their cards and place all their hands. The same rules apply for the banker as well. 
Hands
•	Each player must be granted accommodation on the table. This will be their section. There must be room for three piles of cards which will represent the 3 hands as follows: 
o	1st Hand is 3 cards and placed in front furthest away from the player,
o	2nd Hand is 5 cards, aligns vertically from the 1st Hand and is placed closer to the player; this is the middle pile, 
o	3rd Hand is also 5 cards and placed closest to the player and is aligned to the other 2 hands/piles.
•	1st hand must be weaker than the 2nd hand which must be weaker than the 3rd hand. Failure to follow this order eliminates you and you lose.
•	These hands are poker type hands and are rated smallest to largest as follows:
1.	High Card, 
2.	One pair, 
3.	Two pairs, 
4.	Three of a kind, 
5.	Straight (5 cards in consecutive order regardless of suit), 
6.	Flush (5 cards in any order but of the same suit, 
7.	Full house (three of a kind and a pair).
These can be combined but the weakest hand must be in front, with a stronger hand in the middle and the strongest at the back. The value of the cards matter; a pair of Aces is larger than a pair of queens and a straight flush from 2-6 is smaller than 3-7 and so on. Having an Ace always makes your hand stronger so that ace to 5 is stronger than 9 to king. However, it is weaker than 10 to Ace. 
A ‘3-card’ straight or a ‘3-card’ flush has no value unless the player has a ‘Special’ whereby all his hands are straights for the 3-card straight to matter or all his hands are flushes for the 3-card flush to matter. For all other circumstances, the 3-card straight and the 3 card-flush are both irrelevant.
•	Suit also matters and are ranked from highest to lowest as follows: 
1.	Hearts, 
2.	Spades, 
3.	Diamonds,
4.	Clubs.
Specials
There are also specials in which the player needs to declare and if so, does not participate any further in that round. There must be a single space in each player’s section, including the banker, that can accommodate a single pile of cards. This is the space where the entire pile belonging to the player/banker is placed when they are dealt a Special. These specials, ranked from lowest to highest, are as follows: 
1.	No face (no jacks, queens or kings), 
2.	Four of a kind (four cards of the same value, for example 4 kings, 4 8s, 4 aces etc), 
3.	Straight Flush (5 or more cards in consecutive order belonging to the same suit), 
4.	Straight-straight-straight (all three hands are straights. A ‘straight’ consists of cards of a consecutive order in any suit. This special is when the 3 hands are straights), 
a.	It is the only event that gives a 3-card straight of the 1st hand any relevance. Otherwise, a 3-card straight does not matter.
5.	Flush-flush-flush (all hands are cards of the same suit), 
a.	It is the only event that gives a 3-card flush of the 1st hand any relevance. Otherwise, a 3-card flush does not matter.
6.	Royal Flush (10 to Ace of same suit), 
7.	6 1/2 (6 pairs and one other card or 5 pairs and one three of a kind. Four of a kind could be split into pairs to accommodate this special), 
8.	Full Suit (All the cards of the same suit).
When a player is dealt ‘a special’, they stack all their cards together and place them face down with the top card being flipped face up. If the Banker has a higher special than this player then the player loses and must pay the banker.
The player and the banker must be given the opportunity to declare the type of special. If they declare the wrong special they lose the round.

Going Live on DigitalOcean
Step 1 — Get a Droplet
1.	Sign up at digitalocean.com
2.	Create Droplet → Ubuntu 22.04 → Basic → $12/month (2GB RAM)
3.	Add your SSH key during setup

Step 2 — Domain
1.	Buy vurglife.com at namecheap.com (~$12/year)
2.	In Namecheap DNS settings, point A record to your Droplet IP
3.	Add www CNAME → vurglife.com

Step 3 — Server Setup (run on Droplet)
# Update system
sudo apt update && sudo apt upgrade -y

# Install Node.js 20
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt install -y nodejs

# Install PM2 (keeps server running)
sudo npm install -g pm2

# Install Nginx
sudo apt install -y nginx

# Install Certbot (free SSL)
sudo apt install -y certbot python3-certbot-nginx

Step 4 — Upload your code
# From your Windows machine (using Git)
git init
git add .
git commit -m "VurgLife v1"
git remote add origin https://github.com/YOUR_USERNAME/vurglife.git
git push -u origin main

# On Droplet
git clone https://github.com/YOUR_USERNAME/vurglife.git
cd vurglife/vurglife-platform
npm install
cp .env.example .env
nano .env   # Fill in production values

Step 5 — Nginx config
sudo nano /etc/nginx/sites-available/vurglife
Paste:
server {
    listen 80;
    server_name vurglife.com www.vurglife.com;

    location / {
        proxy_pass http://localhost:3000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_cache_bypass $http_upgrade;
    }

    location /ws {
        proxy_pass http://localhost:3001;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "Upgrade";
        proxy_set_header Host $host;
    }
}
sudo ln -s /etc/nginx/sites-available/vurglife /etc/nginx/sites-enabled/
sudo nginx -t
sudo systemctl restart nginx

Step 6 — SSL
sudo certbot --nginx -d vurglife.com -d www.vurglife.com

Step 7 — Start with PM2
pm2 start server/index.js --name vurglife-platform
pm2 start ../poker-server/index.js --name sipsam-game
pm2 save
pm2 startup

Step 8 — Email

2.	Add vurglife.com as a domain, verify DNS
3.	Copy API key to .env → RESEND_API_KEY
