---
name: modernization-scout
description: Use when recommending current AI, automation, model, API, hosting, security, analytics, database, testing, or deployment improvements for PokerProject.
---

For modernization recommendations:

1. Confirm the business goal and affected user group.
2. Verify current official sources when the recommendation depends on changing models, APIs, pricing, platform features, security rules, or deployment options.
3. Compare build cost, runtime cost, AI/token cost, maintenance cost, security risk, and user value.
4. Prefer incremental changes that can be validated and rolled back.
5. Record accepted decisions in `docs/system-development/DECISIONS.md` and deferred ideas in `ROADMAP.md`.

Do not recommend a tool because it is new. Recommend it only when it improves the project outcome.
